## 这游戏玩什么

你是一个有伴侣的普通人。每天早上领到一笔买菜钱（8~28元，够紧的），出门走到菜市场，在有限时间里买到尽可能好的食材，回家做饭，端上桌。

菜场不是超市——摊主有自己的性格和心情，菜有好有坏有暗坑，砍价看人情，天气影响品种和价格，时间不够就得赶。做菜不是点按钮——你真的要一步步写：切番茄、热锅、倒油、炒蛋、盛出、再炒番茄、放回去、加盐、出锅。做得好她夸你，做砸了她也吃。

**她不是 NPC**——你的伴侣是真人。游戏不编她的口味、不替她反应。她说好吃就是好吃，她说太咸你就得记着下次少放盐。

## 快速开始

### 方式一：纯 Python 终端运行

```bash
# 不需要安装任何依赖，纯 Python 标准库
git clone https://github.com/cookies367/shangzhuochifan.git
cd shangzhuochifan

# 校验数据（可选）
python validate_data.py

# 开始玩
python -c "
from market_engine import MarketGame
g = MarketGame()
print(g.cmd('新局'))
print(g.cmd('菜场'))
print(g.cmd('去 veg_1'))
print(g.cmd('买 番茄'))

```

## AI 厨师系统（V3 独家）

这是 V3 最大亮点——你不再需要逐条写做菜指令，**让 AI 角色帮你做饭**。

### 使用方式

```
厨师 做晚饭
AI乱做
AI做饭 西红柿鸡蛋
```

### AI 厨师的工作流

1. **查看冰箱**：AI 自动检查冰箱里有什么食材
2. **判断心情**：每日随机决定烹饪态度——认真/随意/醋意/恶搞
3. **自由发挥**：AI 根据食材和心情决定做什么菜、怎么做
4. **产出评价**：根据态度和食材品质，产出不同品质的菜品
5. **厨艺成长**：非恶搞态度必定提升厨艺和熟练度

### 烹饪态度影响

| 态度 | 品质倾向 | 说明 |
|------|---------|------|
| 😊 认真 | 中高 | 规规矩矩做菜，保底成功 |
| 😌 随意 | 中低 | 凭感觉来，偶尔翻车 |
| 😏 醋意 | 随机 | 心情不好，可能放奇怪配料 |
| 👻 恶搞 | 低~黑暗 | 纯搞事，可能产出黑暗料理，不涨厨艺 |

## 游戏流程详解

### 一局的结构

```
领预算 → 出门路途 → 逛菜场（限时）→ 回家 → 做菜 → 端上桌 → 她说
  ↑                                                    ↓
  └────── 新一天，冰箱里的剩菜可能过期 ←────────────────┘
```

### 领预算

每天早上随机领 8~28 元。紧巴巴的——买好菜就得砍价，不砍价就得多走几家比价。

### 出门路途

根据天气不同，出门的体验不同：
- 晴天：溜达过去，袋子晃来晃去
- 阴天：风往领口里钻
- 雨天：鞋踩进水坑，袋子勒手指

### 逛菜场

菜场有 **16 个固定摊位**，分 6 个区：

| 分区 | 摊位 | 卖什么 |
|------|------|--------|
| 蔬菜区 | 王姐蔬菜、老张菜摊、小陈鲜蔬 | 小白菜、菠菜、生菜、茼蒿… |
| 根茎区 | 刘姐根茎、赵哥山货 | 土豆、萝卜、山药、春笋… |
| 瓜果区 | 孙姐瓜果、阿亮果蔬 | 番茄、黄瓜、茄子、青椒… |
| 豆菌区 | 周婶豆角、老吴菌菇 | 豆角、香菇、平菇、金针菇… |
| 肉鱼区 | 胖哥肉铺、李姐鲜肉、老何鱼摊、小黄水产 | 五花肉、排骨、鲫鱼、虾… |
| 蛋品区 | 马婶蛋铺 | 鸡蛋、鸭蛋… |
| 干货区 | 老方干货 | 虾皮、紫菜、干辣椒… |

还有 **3 个流动摊**（随机出现）：
- 老农自种菜（便宜但量少）
- 散市尾货（快收摊了，甩卖）
- 时令限定（按季节卖香椿、荠菜、莲蓬、板栗等别处没有的）

### 时间系统

菜场有营业时间，不同时段时间点数不同：

| 时段 | 时间点数 | 特点 |
|------|---------|------|
| 早市 | 8点 | 货全但人多 |
| 上午 | 5点 | 正常 |
| 散市 | 3点 | 便宜但缺货多 |

每个操作消耗时间：逛摊 -1、砍价 -1、聊天 -1、细看 -0.5。时间归零强制回家。

### 买与砍价（V3 重构）

看到菜可以直接买。价格受季节、天气、时段影响——当季便宜，反季贵；雨天涨价；散市甩卖。

**砍价（V3 新逻辑）**：`砍价 菜名 [话术]`——砍价成功**不会自动购买**，只记录价格。之后 `买` 时使用砍低价。

- 砍价话术影响成功率："老客户了" → 熟客加分、"隔壁更便宜" → 竞争策略（有风险）、"多买点便宜点" → 量大策略
- 支持 `砍价 只谈价`：只砍价不买，方便囤积砍低价
- 砍价缓存机制：同一摊位同一食材砍价后，当日购买自动享受砍低价

摊主 5 种性格影响砍价：
| 性格 | 砍价难度 |
|------|---------|
| 实在 | 容易，价格本来就实 |
| 爽快 | 一般 |
| 话唠 | 聊着聊着就便宜了 |
| 算计 | 难，算得比你精 |
| 死硬 | 最难，一分不让 |

### 品质与暗坑

菜的品质分五档：

| 品质 | 标记 | 说明 |
|------|------|------|
| 优品 | ★优 | 水灵、新鲜 |
| 不错 | ○好 | 正常水平 |
| 一般 | ～般 | 凑合能用 |
| 不太行 | ✗差 | 发蔫、有斑点 |
| 暗坑 | ！坑 | **表面看不出来，切/洗时才暴露** |

暗坑是核心机制——你看不出菜有问题，买回家切开才发现烂心、注水、里面发黑。细看（`细看 菜名`）可以提前发现，但消耗时间。

### 熟客系统

反复去同一个摊，熟客等级会提升：

| 等级 | 条件 | 权益 |
|------|------|------|
| 1级 面熟 | 3次 | 摊主认识你 |
| 2级 常来 | 5次 | 报价不虚高 |
| 3级 老客 | 8次 | 偶尔多送一点 |
| 4级 赊账 | 12次 | 钱不够也能买 |

### 天气与灾害

每天随机天气（晴/阴/雨），影响：
- 雨天：涨价 10%、少带 1 样菜、少 1 点时间、摊位可能关门
- 5 种随机灾害：暴雨淹市、检查来了、摊主缺席、批发涨价、好货到货、节前抢购

### 做菜

回家后开始做菜。三种方式：

**快做**——一句话描述做法：
```
做法 番茄切块，鸡蛋打散先炒盛出，再炒番茄出汁放回蛋，加盐出锅
```

**细做**——一步一步写：
```
做 番茄炒蛋
番茄切块
鸡蛋打散，热锅倒油先炒蛋盛出
炒番茄出汁，放回鸡蛋翻炒
加盐出锅
```

**AI 厨师（V3 新增）**——让 AI 角色自动发挥：
```
厨师 做晚饭
AI乱做
```

做菜引擎追踪：
- **熟度 0-100**：从生到糊，每种食材有不同的最佳区间
- **感官反馈**：番茄从鲜脆到微皱出水，鸡蛋从流动到膨胀定型
- **厨房意外**：筷子掉地上、锅铲滑手、盐罐倒多了……
- **配方发现**：番茄+鸡蛋下锅会提示"汁水裹着蛋花，这搭配太对了"，解锁菜谱

### 端上桌

菜做好端上桌那一刻——颜色、气味、她夹一筷子的反应。品相从"绝了"到"不忍直视"五档，受食材品质、烹饪过程、厨房意外综合影响。

### 口味记忆

你可以告诉她爱吃什么、不吃什么：
```
记得 她爱吃虾
记得 她不吃香菜
记得 土豆要脆的
```

### 冰箱与过期

买多了没做的食材留在冰箱里，但会过期：
- 绿叶菜 1-2 天
- 根茎 7-14 天
- 肉 1-2 天
- 干货 30-60 天

每天早上开新局时会提示哪些东西过期了。

## 引擎架构（V3）

```
MarketGame
├── new_day()        # 新一天：roll天气/预算/灾害/缺货/摊主状态
├── cmd(instruction) # 指令入口，解析并分发
├── visit_stall()    # 逛摊：展示菜品、好感、故事、互动
├── go_home()        # 回家：路途描写、预处理、菜谱提示
├── cook_step()      # 做菜一步：解析动词/食材、推进熟度、感官反馈
├── _quick_cook()    # 快做：自动拆分步骤、关键时刻暂停
├── _serve()         # 端上桌：品相评分、叙事、她的话
├── save()/load()    # 存档（JSON）
├── ChefEngine       # 🆕 AI 厨师系统
│   ├── cook()       # AI 自由发挥做菜
│   ├── _get_mood()  # 判断心情和烹饪态度
│   └── _evaluate()  # 评价菜品品质
└── game_wrapper     # 🆕 Operit 插件桥接
    ├── play()       # 启动游戏
    └── process()    # 处理指令
```
